# Prototipo de sistema de alerta temprana para predecir la deserción en Educación Media Superior usando XGBoost, SHAP y LLMs

## Estructura del Repositorio

app/: Carpeta para la interfaz de usuario desarrollada en Streamlit.

data/: Directorio destinado al almacenamiento del dataset sintético y archivos CSV.

notebooks/: Espacio para los cuadernos de Jupyter (.ipynb) con el modelado de XGBoost y análisis con SHAP.

README.md: Descripción general del proyecto y su organización.
